# ADR-002: Топологія VictoriaMetrics (Single-node)

- Статус: Accepted
- Дата: 2026-03-07

## Контекст
На поточному етапі потрібен production-ready мінімум із низькою складністю підтримки.

## Рішення
Використовуємо `VictoriaMetrics single-node`.

## Обґрунтування
- Мінімальний час впровадження
- Проста експлуатація для невеликої DevOps-команди
- Достатньо для стартового обсягу метрик

## Наслідки
- Потрібен регулярний backup volume
- Cluster-міграція переноситься у post-prod етап
